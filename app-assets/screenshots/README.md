# Every Wall Falls — App Store Screenshots

Marketing screenshots for the App Store listing, captured from live gameplay and
framed in the game's own style (engraved bronze-on-stone headlines).

Live copies: `https://bagelarcade.com/app-assets/screenshots/<filename>`

## Size

All files are **1290 × 2796** — Apple's **iPhone 6.9"** display class
(iPhone 16 Pro Max / 15 Pro Max). This is the required size in App Store Connect;
Apple auto-scales it down to the 6.5" class, so this one set covers all modern
iPhones. (If App Store Connect ever asks for a 6.5" set specifically, the same
images are accepted.)

## raw/

The unframed device captures (also 1290 × 2796), in case you'd rather upload

## Notes

- Apple allows up to 10 screenshots; 3–5 is the sweet spot, so this set is ready as-is.
- Google Play uses a different spec (min 2, 16:9 or 9:16, up to 8) — these portrait
  images work there too; Play is more lenient on exact pixel size.
- To regenerate or re-word headlines, see `../_source/` (`compose.js` builds these
  from the raw captures; edit the `screens` array to change captions).
